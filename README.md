# Welcome to Sarasavi Library Management System

A Windows desktop application to manage day to day library operations such as book cataloging, member management, and book issue/return workflows.

---

## Project Info

- **Platform:** Windows Desktop Application  
- **Technology:** .NET Framework (WinForms/WPF)  
- **IDE:** Visual Studio  
- **Language:** C#  

> Update these lines if your project uses WinForms or WPF specifically, and mention the exact .NET Framework version (e.g., 4.7.2 / 4.8).

- Issue & Return Books
